# Bank-Account-Management-System
Bank Account Management System (Python)

Project Description

This is a console-based Bank Account Management System developed using **Python and Object-Oriented Programming (OOP) principles.
The system allows users to create bank accounts, deposit and withdraw money, check balances, and transfer funds securely between accounts.

This project demonstrates core programming concepts such as:

* Classes and Objects
* Methods and Encapsulation
* Conditional Logic
* User Input Handling
* Dictionary-based data storage

---

Features

* Create new bank accounts
* Deposit money
* Withdraw money
* Check account balance
* Transfer money between accounts
* Input validation and error handling
* Prevents invalid and duplicate account operations

---

 Technologies Used

* Programming Language: Python 
* Paradigm: Object-Oriented Programming (OOP)
* **Interface:** Command Line / Terminal
